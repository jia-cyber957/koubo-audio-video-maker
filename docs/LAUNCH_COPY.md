## 中文宣传文案

花生AI这类口播成片工具确实好用，但对个人创作者来说，价格有点肉疼。

所以我做了一个“开源低成本版口播成片”工作流。你也可以口头叫它“花生AI青春平替”，但它不是官方青春版，也不是花生AI的附属项目。

它不是一键黑箱工具，而是把口播配画面拆成几个可检查步骤：

1. 输入一段口播音频。
2. 火山引擎转写出字级时间轴。
3. AI 按文本语义切成 10 秒以内的分镜。
4. 去 Pexels/Pixabay 多轮找免费素材。
5. 让 LLM 根据口播文本、关键词、标题和 tags 亲自评分。
6. 达标的点停止，没达标的点继续补。
7. 下载素材并装配成可审片时间线。

主打就是低成本和可控。可以配合 OpenCode 里的免费模型做素材审核，整体费用主要取决于转写和 API 使用。

它适合做知识类、讲述类、口播类短视频。不保证一键完美，但每一步都有产物，能检查，能返工，能恢复。

## B 站/动态短文案

花生AI太贵？我做了一个开源低成本口播配画面工作流。

输入音频后，它会转写、按语义切分镜、去免费素材站找视频，再让 AI 根据文案和候选信息打分。重点是：不是黑箱一键生成，而是每一步都能看、能改、能接着跑。

适合想省钱、想自己控制画面质量的人。非官方项目，不是花生AI青春版，只是一个预算友好的开源替代路线。

## 不建议使用的标题

- 花生AI青春版
- 花生AI开源版
- 花生AI免费版
- 花生AI替代官方版
- 花生AI破解版

这些写法容易让人误会你和花生AI有官方关系，或者让人以为你在复刻/冒充它的产品。
