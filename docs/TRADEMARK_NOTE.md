# 命名和碰瓷边界

你可以碰瓷热点，但不要把别人的品牌名当成自己的项目名。

## 相对稳的表达

- 花生AI太贵？我做了一个开源低成本口播成片工作流。
- 面向个人创作者的低成本口播配画面方案。
- 一个预算友好的开源替代路线。
- 非官方项目，不隶属于花生AI。

## 不建议的表达

- 花生AI青春版
- 花生AI开源版
- 花生AI免费版
- 花生AI官方平替
- 花生AI破解版

## 为什么

这些名字容易让用户误以为项目和花生AI存在官方关系、授权关系或从属关系。更稳的做法是：在正文里提“因为花生AI这类工具太贵，所以做了低成本开源方案”，但项目名、仓库名、logo 不使用对方品牌。

```text
koubo-audio-video-maker
```
